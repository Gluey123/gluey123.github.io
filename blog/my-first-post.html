My First Post

This is where the post starts. Replace this with whatever you actually want to say. It doesn't need to be long, it doesn't need to be perfect, it just needs to exist.

Writing things down is underrated. Even if no one reads it, the act of putting thoughts into sentences forces you to figure out what you actually think.

A quick note on structure

Each post lives in its own file inside the blog/ folder. To write a new one, copy post-template.html, rename it, fill it in, and add a link to blog.html. That's it.
